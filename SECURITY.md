# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this plugin, **please do not file a public issue.**

Instead, report it privately by emailing **johnboyce@comcast.net** with:

- A description of the vulnerability
- Steps to reproduce (if applicable)
- The version of the plugin affected
- Any relevant logs or screenshots

You should receive a response within 48 hours acknowledging receipt. We will work with you to understand the issue and coordinate a fix before any public disclosure.

## Scope

This policy covers the OpenSpec IntelliJ plugin source code and its published artifacts on the JetBrains Marketplace. It does not cover the upstream [OpenSpec CLI](https://github.com/fission-ai/openspec) — please report vulnerabilities in the CLI to the Fission AI team directly.

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest release | Yes |
| Older releases | Best effort |
