# Test Domain

## Domain
Test domain for unit testing.

## Requirements

### Requirement: Basic Feature

The system SHALL provide a basic feature.

**Keyword:** SHALL

#### Scenarios

**Scenario: Feature works**
- GIVEN the system is running
- WHEN the user invokes the feature
- THEN the system SHALL respond correctly

### Requirement: Optional Feature

The system MAY provide an optional feature.

**Keyword:** MAY

#### Scenarios

**Scenario: Optional feature present**
- GIVEN the system is configured with optional features
- WHEN the user looks for the feature
- THEN it MAY be available
