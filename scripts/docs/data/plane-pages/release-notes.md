# Release Notes

## v0.1.0 — Ship It Clean (Target: April 2026)

First public release focused on a stable, polished first impression.

### Features
- OpenSpec CLI integration with auto-detection
- Spec tree browser in tool window
- Workflow panel with interactive pipeline
- Change lifecycle actions (Propose, Generate, Apply, Archive)
- Direct API support for Claude, OpenAI, and Gemini
- Delivery-aware Generate All button
- Forgejo issue tracking integration
- Plane work item tracking integration
- Settings panel with organized sections
- Spec validation inspections

### Infrastructure
- Automated issue lifecycle (create/update trackers on propose/apply/archive)
- Setup scripts for Forgejo and Plane workspace configuration
- Copilot and Claude Code skill definitions

## v0.2.0 — Spec Intelligence (Target: May 2026)

Features that differentiate from all existing OpenSpec tools.

### Planned
- Code-to-spec traceability via gutter markers
- Spec coverage indicators
- Inline quick-fixes for spec violations
- Change diff viewer
- Spec search

Community adoption and ecosystem integration.
