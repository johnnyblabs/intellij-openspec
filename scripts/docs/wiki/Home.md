# OpenSpec IntelliJ Plugin

Welcome to the **OpenSpec IntelliJ Plugin** wiki — the comprehensive reference for the spec-driven development plugin for IntelliJ IDEA.

OpenSpec brings structured, spec-driven development directly into your IDE with a hybrid CLI/built-in engine, AI-powered artifact generation, and real-time validation.
